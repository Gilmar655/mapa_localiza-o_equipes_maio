# Site – Acompanhamento de Equipes em Campo | Região Leste

Arquivo principal: `index.html`

Como publicar no Netlify:
1. Acesse https://www.netlify.com/
2. Entre na conta.
3. Arraste a pasta deste projeto para a área de deploy.
4. O Netlify criará um link público automaticamente.

Observação: este site usa mapa via OpenStreetMap/Leaflet e os dados do Excel foram incorporados no `index.html`.
Para atualizar os pontos, substitua a base `DADOS` no final do arquivo ou gere novamente o site com o Excel atualizado.
